## 🚀 Featured Project: Instant Arena

A production-grade 3D first-person shooter built entirely in the browser with Three.js.

### What Makes This Special

This isn't just a game—it's a **complete engineering demonstration**:

- ✅ **Zero external assets** — Procedural audio, procedural geometry
- ✅ **AI state machine** — Tactical 3-state opponent
- ✅ **Single-file architecture** — 2,500 lines of organized, production-ready code
- ✅ **Performance optimized** — 60 FPS target on mid-range hardware
- ✅ **Zero build step** — Direct browser execution

### Technical Highlights

| Component | Implementation |
|-----------|----------------|
| **Graphics** | Three.js (r163), WebGL 2.0 |
| **Audio** | Web Audio API — procedural sound generation |
| **AI** | Finite state machine (PUSH → ENGAGED → RETREAT) |
| **Physics** | Custom collision detection, raycasting |
| **Input** | Pointer Lock API, keyboard mapping |

### Quick Start

```bash
git clone https://github.com/dshirmoh-sketch/portfolio.git
cd portfolio

# Python 3
python3 -m http.server 8000

# Or Node.js
npx serve .

# Then open http://localhost:8000
```

Click **"Deploy To Arena"** to play.

### Controls

| Input | Action |
|-------|--------|
| Mouse | Look around |
| W/A/S/D | Move |
| Left Click/Space | Fire |
| 1/Q | Pulse Carbine |
| 2/E | Heavy Bolter |
